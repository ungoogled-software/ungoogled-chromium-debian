
# Installation

<br>

*instructions on how to set up your system.*

## Steps

*Once you have set up your system,* <br>
*you can follow up with these step.*

<br>

1.  **Update your system**

    ```shell
    sudo apt update
    ```
    
<br>

2.  **Install Ungoogle Chromium**

    ```shell
    sudo apt install -y ungoogled-chromium
    ```
